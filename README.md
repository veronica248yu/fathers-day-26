# Father's Day

A single-page interactive Father's Day site. Hover a flower and it grows into
its vegetable; click the vegetable to return it to the flower.

Static HTML/CSS/JS — no build step. To run locally, open `index.html` in a
browser or serve the folder with `python -m http.server`.

## Hosting on GitHub Pages

Upload the contents of this folder to a GitHub repository, then enable
**Settings → Pages → Source: `main` branch / `/ (root)`**. The site will be
live at `https://<your-username>.github.io/<repo-name>/`.
